## Taskys 在线便笺

### 在线链接
http://note-andluo.top

### 技术栈
1. node/express/ejs/oauth2/webpack/less/sqlite3
2. MVC设计模式/模块化开发/发布订阅模式/瀑布流布局

### 功能点
1. 集成第三方(github)的登录功能
2. 可以增加、修改、删除便笺
3. 可以对便笺重要性打分

### 功能介绍
1. 为每一段工作时间设置定时
2. 为每天要达到的任务做个记录
3. 统计自己的目标量/工作量

### 待完成
1. 排序功能，按重要性排序，按日期排序
2. 集成其它第三方登录
3. 增加更多的动画效果
